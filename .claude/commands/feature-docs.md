---
description: Show available feature documentation from docs/features/
---

List all available feature documentation in the docs/features/ folder.

Steps:
1. List files: `ls docs/features/`
2. Display the available documentation files with dates
3. If user wants to read a specific doc, show the content
4. If $ARGUMENTS is provided, read that specific feature doc

Example:
- `/feature-docs` - List all available docs
- `/feature-docs cms` - Show CMS feature documentation
