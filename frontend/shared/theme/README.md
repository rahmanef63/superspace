## Theme Selector Bundle

This folder contains a **self‑contained copy** of all theme selector–related files you need to move into another project.

You can:

- Copy the entire `theme-selector-bundle` folder into your new repo.
- Let an AI assistant (or yourself) move the contents into the final structure (e.g. into `src/components`, `src/hooks`, `public/r`, etc.) and fix import paths.

### Included structure

- `./components/layout/active-theme.tsx`
- `./components/layout/theme-selector.tsx`
- `./components/providers/theme-provider.tsx`
- `./config/theme.ts`
- `./hooks/themes.ts`
- `./store/theme-preset-store.ts`
- `./utils/theme-fonts.ts`
- `@/public/r/registry.json`

Refer back to `THEME_SELECTOR_SETUP.md` at the project root for detailed wiring instructions.
